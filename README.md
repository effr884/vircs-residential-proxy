## 住宅代理IP是什么？为什么机房 IP 不够用？

**住宅代理IP（Residential IP）** 是指由互联网服务提供商（ISP）分配给普通家庭宽带用户的真实 IP 地址。它的对立面是数据中心 IP（Datacenter IP），也就是我们常见的 VPS/云服务器 IP。

区别在哪里？一句话说清楚：平台风控系统能识别 IP 的来源。

机房 IP 来自 ASN 归属于数据中心的地址段，Facebook、TikTok、Google 这些平台的反欺诈系统都有能力实时扫描。一旦判断你是从数据中心访问，立刻触发短信验证、行为验证，严重的直接封号。

住宅 IP 不一样。它归属于真实的 ISP（比如 AT&T、Comcast），行为特征和普通网民一模一样，平台没法轻易区分。

所以在跨境电商选品调研、TikTok 多账号运营、社媒矩阵管理、数据采集这些场景里，**住宅代理IP推荐**的核心标准只有一个：IP 来源是否真实、干净、可信。
